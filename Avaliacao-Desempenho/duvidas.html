Dúvidas Frequentes

Primeiros passos.

Primeiramente deve-se cadastrar a Cargo dos colaboradores, após já pode-se cadastrar os COLABORADORES. Após os primeiros cadastros já pode-se iniciar os cadastros de CRITÉRIOS e AVALIAÇÃO.

Como faço para ALTERAR um registro?

Para alterar um registro, basta ir em CONSULTA, selecionar a tabela que contém o registro a ser alterado e CLICAR NA LINHA onde o registro se encontra, abrirá uma janela com os dados para serem alterados. É só alterar os dados e clicar em CONFIRMAR.

Como faço para APAGAR um registro?

Não é possível apagar um registro por completo, por motivo de integridade do banco de dados, o que pode ser feito é INATIVAR um colaborador, é só seguir os mesmos passos para ALTERAR REGISTRO, basta ir em CONSULTA, selecionar a tabela que contém o registro a ser alterado e CLICAR NA LINHA onde o registro se encontra, abrirá uma janela com os dados para serem alterados ou inativados. É necessário marcar a opção ATIVO como NÂO e CONFIRMAR.
